  • Stap 5.8: DHCP Gedetailleerde Configuratie

    DHCP (Dynamic Host Configuration Protocol) beheert automatisch IP-adres toewijzing en netwerkconfiguratie.

    üìä DHCP Lease Process

    1. DHCPDISCOVER: Client zoekt DHCP server
    2. DHCPOFFER: Server biedt IP-adres aan
    3. DHCPREQUEST: Client accepteert aanbod
    4. DHCPACK: Server bevestigt toewijzing
    5. DHCPRELEASE: Client geeft IP-adres vrij (optioneel)

    üõ°Ô∏è DHCP Security Features

    Security Feature Beschrijving Voordeel
    MAC Filtering Alleen specifieke MAC adressen krijgen IP Voorkomt onbevoegde toegang
    Name Protection Beschermt tegen DNS spoofing Voorkomt man-in-the-middle attacks
    Conflict Detection Controleert op IP-conflicten Garantieert unieke IP-adressen
    DHCP Audit Logging Gedetailleerde logging van alle acties Forensische analyse mogelijk

    üîÑ DHCP Failover

    • Load Balance Mode: 50/50 verdeling tussen servers
    • Hot Standby Mode: Primaire server actief, secundaire standby
    • State Switchover: Automatische failover bij uitval
    # DHCP failover configuratie
Add-DhcpServerv4Failover -Name "DHCP-Failover" -PartnerServer "LAB-DC02.lab.local" -ScopeId 10.0.0.0 -LoadBalancePercent 50 -MaxClientLeadTime 1:00:00 -AutoStateTransition $true

# DHCP security instellingen
Set-DhcpServerv4FilterList -Allow $true -List "08-00-27-00-00-01"
Set-DhcpServerAuditLog -Enable $true -MaxMBFileSize 70 -Path "C:\Logs\DHCP"

# DHCP conflict detection
Set-DhcpServerv4OptionValue -ScopeId 10.0.0.0 -Router 10.0.0.1 -DnsServer 10.0.0.1 -DnsDomain "lab.local"
Set-DhcpServerSetting -ConflictDetectionAttempts 2

# DHCP reservations voor servers
Add-DhcpServerv4Reservation -ScopeId 10.0.0.0 -IPAddress 10.0.0.10 -ClientId "08-00-27-00-00-02" -Description "File Server"

21 Group Policy Management

‚ÑπÔ∏è
Doel: Group Policy is een krachtig hulpmiddel voor centraal beheer van gebruikers- en computerinstellingen in een Active Directory domein.

Group Policy Objects (GPOs) bevatten instellingen die worden toegepast op gebruikers en computers. Ze worden opgeslagen in Active Directory en kunnen worden gekoppeld aan sites, domeinen of organizational units (OU's).

Type Beschrijving Scope
